 # TODA MOEDA APP

<p> Aplicativo de conversão de moedas desenvolvido utilizando o framework Flutter</p>

<h1> Informações </h1>


<p> Neste projeto foi contruindo um App que faz conversão de moedas de forma dinâmica. Para obter os dados atualizados dos valores das moedas, fou utilizado um API que pode ser obtida através do seguinte link: <a href= ' https://hgbrasil.com/status/finance' ></a>, essa API fornece dados no formato Json. </p>
